# Google Chrome Instructions

## Method 1: Allow Ads Through the Extension Icon

This is usually the easiest and most convenient method.

1. Open Shiguangji in Google Chrome.
2. Click the **Extensions** icon in the upper-right corner of the browser.
3. Find the ad-blocking extension you are using.
4. Click the extension icon to open its control panel.
5. Select an option such as:
   - Pause on this site
   - Allow ads on this site
   - Trust this site
   - Add to allowlist
6. Refresh the page.

The exact wording may vary depending on the extension, but most ad blockers provide an option to disable blocking for the current website only.

---

## Method 2: Temporarily Disable the Extension

If you are not sure which extension is blocking the ads, you can temporarily disable the relevant extension.

1. Open Google Chrome.
2. Click the three-dot menu in the upper-right corner.
3. Select **Extensions**.
4. Open **Manage Extensions**.
5. Find the ad-blocking extension.
6. Turn off the extension using the switch.
7. Return to Shiguangji and refresh the page.

You can also enter the following address directly into the browser:

`chrome://extensions/`

Then press Enter.

Please note that this method disables the extension on all websites until you turn it back on.

# Instructions for Common Ad-Blocking Extensions

## AdBlock

1. Open Shiguangji.
2. Click the **AdBlock** icon in the Chrome toolbar.
3. Select the option that allows ads on the current website.
4. Confirm the change if prompted.
5. Refresh the page.

Depending on the version, the option may appear as:

- Allow ads on this site
- Pause on this site
- Always allow ads on this site
- Add to allowlist

Choose the option that applies only to the current website.

---

## Adblock Plus

1. Open Shiguangji.
2. Click the **Adblock Plus** icon in the browser toolbar.
3. Open the extension settings.
4. Find the list of allowed websites.
5. Add the Shiguangji domain.
6. Save the setting and refresh the page.

Some versions also provide a switch labeled:

**This website**

or:

**Block ads on this website**

Turn off blocking for the current website only.

---

## uBlock Origin

1. Open Shiguangji.
2. Click the **uBlock Origin** icon in the Chrome toolbar.
3. Click the large power button in the extension panel.
4. Confirm that the power button changes to the disabled state.
5. Refresh the page.

This setting normally applies only to the current website.

Click the power button again if you later want to restore blocking on Shiguangji.

---

## uBlock Origin Lite

1. Open Shiguangji.
2. Click the **uBlock Origin Lite** icon.
3. Find the filtering level for the current website.
4. Change the filtering setting to:

**No filtering**

5. Refresh the page.

This change normally applies only to the current website and does not affect filtering on other websites.

---

## AdGuard AdBlocker

1. Open Shiguangji.
2. Click the **AdGuard** icon in the Chrome toolbar.
3. Find the option related to filtering on the current website.
4. Turn off filtering for Shiguangji.
5. Refresh the page.

You may also open the AdGuard settings and add Shiguangji to the:

**Allowlist**

After that, AdGuard will continue working normally on other websites.

---

## Ghostery

1. Open Shiguangji.
2. Click the **Ghostery** icon in the browser toolbar.
3. Choose to pause protection or trust the current website.
4. Select a permanent option such as:

**Always**

or:

**Always trust this site**

5. Refresh the page.

Ghostery will continue protecting you on other websites while allowing Shiguangji to display ads.

## Built-In Browser Ad Blocking

Some Chromium-based browsers include their own advertising, tracking, or privacy protection features.

If you have not installed an ad-blocking extension but still see a warning that ads are blocked, look near the address bar for:

- An ad-blocking notice
- A shield icon
- A privacy protection icon
- A tracking protection message
- A blocked content notification

Click the relevant icon and allow advertising or disable protection for the current website, then refresh the page.

# Still Not Working?

If you have already added Shiguangji to your allowlist but the website still reports that ads are blocked, try the following steps.

## 1. Refresh the Page

After changing the settings, perform a full refresh.

On Windows, press:

`Ctrl + F5`

On macOS, press:

`Command + Shift + R`

---

## 2. Check for Multiple Blocking Extensions

Some users have more than one advertising, privacy, script, or tracking extension installed.

Examples include:

- AdBlock
- Adblock Plus
- uBlock Origin
- uBlock Origin Lite
- AdGuard
- Ghostery
- Privacy Badger
- NoScript
- Browser privacy extensions

Even if one extension allows Shiguangji, another extension may continue blocking advertising.

Please check each relevant extension and add the website to its allowlist.

---

## 3. Check Security Software or DNS Filtering

Advertisements may also be blocked by software or services outside the browser.

Common examples include:

- Antivirus web protection
- Router-level ad blocking
- AdGuard DNS
- NextDNS
- Pi-hole
- Proxy or filtering rules
- Company or school network policies
- Privacy-focused DNS services

If you have disabled your browser extensions but the warning remains, one of these services may be responsible.

---

## 4. Clear Cache and Cookies

You may also try clearing the cache and cookies for the current website and then reopening the page.

Please note that clearing cookies may sign you out or reset certain website preferences.
